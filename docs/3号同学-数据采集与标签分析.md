## 数据集分析

### 类别分布

`vs-dataset/` 包含 6 个类别文件夹，对应 6 种中文手语：

| 类别 | 视频数量 |
|------|:--------:|
| 你好 | ~30 个样本 |
| 再见 | ~20 个样本 |
| 对不起 | ~25 个样本 |
| 我爱你 | ~25 个样本 |
| 没关系 | ~20 个样本 |
| 谢谢 | ~30 个样本 |
| **合计** | **约 140 个样本** |

### 标签映射 (`labels.json`)

```json
{
  "0": "对不起",
  "1": "没关系",
  "2": "你好",
  "3": "我爱你",
  "4": "谢谢",
  "5": "再见"
}
```

### 样本多样性

- 每个视频只包含一个完整手语动作。
- 视频格式包括 `.mp4` 和 `.mkv`，时长 2–10 秒不等。
- 样本来自多位不同同学录制，提高模型泛化能力。
- 包含不同角度和距离的样本。

```

    ↓ python extract_dynamic_dataset.py

    ↓ python train_dynamic_gesture_classifier.py
```

## 设计评价

1. **数据质量**：约 140 个样本覆盖 6 类别，平均每类 20+ 样本，对于课程项目而言足够支撑初始模型训练。
2. **格式规范**：视频按类别分文件夹存放，文件名有编号，符合 `extract_dynamic_dataset.py` 的输入要求。
3. **标签清晰**：`labels.json` 提供了简洁的索引→中文标签映射，便于其他同学使用。
4. **后续优化空间**：更多样化的背景、光照条件和更多录制约 50–100 样本/类可进一步提升模型准确率。

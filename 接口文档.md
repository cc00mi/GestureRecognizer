# 接口文档

为了方便协作，建议四个人统一约定下面这些接口。

## 1. 摄像头采集接口（1号提供给2号、4号）

作用：持续输出摄像头图像帧。

```python
# frame_provider.py
def get_frame():
    """
    返回:
        success: bool
        frame: np.ndarray  # BGR格式图像
    """
    pass
```

最简单调用方式：

```python
success, frame = get_frame()
```

如果用类封装，也可以：

```python
class CameraService:
    def read(self):
        """
        返回:
            success: bool
            frame: np.ndarray
        """
        pass
```

约定：

- 图像格式：`OpenCV BGR`
- 分辨率建议统一：`640x480`
- 帧率目标：`15~30 FPS`

## 2. 手势识别接口（2号提供给4号）

作用：输入一帧图像，输出识别结果。

```python
# recognizer.py
def recognize(frame):
    """
    参数:
        frame: np.ndarray

    返回:
        result: dict
        {
            "label_id": 0,
            "label": "谢谢",
            "score": 0.92,
            "handedness": "Right",
            "bbox": [x1, y1, x2, y2],
            "landmarks": [[x, y], [x, y], ...]
        }
    """
    pass
```

最简版本可以只保留：

```python
result = {
    "label": "谢谢",
    "score": 0.92
}
```

建议补充一个“无结果”的约定：

```python
result = {
    "label": "未识别",
    "score": 0.0
}
```

## 3. 标签表接口（3号提供给2号、4号）

作用：维护类别编号与中文名称的映射关系。

建议文件：`labels.json`

```json
{
  "0": "你好",
  "1": "谢谢",
  "2": "喝水",
  "3": "吃饭",
  "4": "回家",
  "5": "1",
  "6": "2",
  "7": "3",
  "8": "4",
  "9": "5"
}
```
